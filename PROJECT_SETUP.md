# Project Setup

## Installation

```bash
pnpm install
pnpm build
```

## Usage

Include the web component in your HTML:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Custom Form Component</title>
</head>
<body>
    <custom-form-component></custom-form-component>
    <script src="./dist/my-form.js"></script>
</body>
</html>
```

## Build Commands

```bash
# Development with watch mode
pnpm dev

# Build components
pnpm build

# Run tests
pnpm test
```
